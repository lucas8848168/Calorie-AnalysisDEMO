## 📝 部署步骤

### 1️⃣ 配置 API 密钥（1 分钟）

打开 `.env.production` 文件，替换你的 API 密钥：

```bash
VITE_DOUBAO_API_KEY=你的真实豆包API密钥
```

**如何获取 API 密钥？**
1. 访问 https://console.volcengine.com/ark
2. 系统管理 → API Key 管理
3. 创建新密钥并复制

### 2️⃣ 推送到 GitHub（1 分钟）

在终端执行：

```bash
# 添加更改
git add .env.production

# 提交
git commit -m "config: 添加 API 密钥配置"

# 推送
git push origin main
```

### 3️⃣ 启用 GitHub Pages（1 分钟）

1. 访问 https://github.com/lucas8848168/Calorie-AnalysisDEMO/settings/pages
2. **Source** 选择 **"GitHub Actions"**
3. 点击 **Save**

### 4️⃣ 等待部署（2-3 分钟）

查看部署进度：
https://github.com/lucas8848168/Calorie-AnalysisDEMO/actions

### 5️⃣ 访问网站 ✨

https://lucas8848168.github.io/Calorie-AnalysisDEMO/

---

## ⚠️ 重要：演示后清理

演示完成后，**立即执行**以下命令删除 API 密钥：

```bash
# 1. 删除包含密钥的文件
git rm .env.production

# 2. 提交
git commit -m "chore: 移除演示配置"

# 3. 推送
git push origin main
```

然后在豆包控制台删除临时 API 密钥。
